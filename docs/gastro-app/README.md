# Gastro-App – KI-Assistent für die Gastronomie

> Du kochst. Die App macht den Rest.

Die Gastro-App ist kein Kassensystem und keine Warenwirtschaft. Sie ist ein
**stiller KI-Assistent** für den Gastronomen: fotografiere den Lieferschein,
der Rest passiert automatisch.

## Kernidee

1. **Beleg fotografieren** → KI liest Artikel, Mengen, Preise
2. **Warenbestand** aktualisiert sich von selbst
3. **Kalkulation** jedes Gerichts passt sich live an den neuen Einkaufspreis an
4. **Vorschläge** kommen zurück: Tagesgericht, Wochenangebot, Preis-Warnung

Ziel: **20 % weniger Stress** im Alltag, bessere Kalkulation, weniger
Verschwendung.

## Leitprinzip

> „Sogar Halil versteht es."

Die App richtet sich an Gastronomen, die in der Küche stehen – nicht am
Schreibtisch. Kein Excel, keine Inventur um Mitternacht, keine
Schätzungen.
